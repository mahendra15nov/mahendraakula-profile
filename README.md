# Mahendra Akula Portfolio

Corporate portfolio website for Mahendra Akula, Senior SDET and Automation Architect.

## Preview locally

Open `index.html` directly, or run any static web server from this folder.

## Deploy on Vercel

1. Create a GitHub repository and push this folder.
2. Import the repository at https://vercel.com/new.
3. Choose **Other** as the framework preset and deploy with no build command.
4. Add a custom domain from the Vercel project settings.

## Contact form

The first submission through FormSubmit will send a confirmation email to activate the form. After activation, future submissions go directly to the configured email address.
